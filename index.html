About

I'm a front-end web developer based in Newcastle, UK — focused on building interfaces that are fast, accessible, and genuinely pleasant to use. My craft is in the front-end, but I have broad knowledge across the stack: from backend PHP and SQL, to UX research, design systems, and API integration.

I currently work at Newcastle University Students' Union, where I design and maintain production websites used by thousands of students every day. I take ownership of features end-to-end — from first commit through to release — working closely with product, design, and content teams.

I graduated from Newcastle University in 2025 with a 2:1 BSc in Computer Science. My final-year dissertation was a human-centred UX research project — 52 participants, think-aloud evaluations, heuristic reviews — resulting in a high-fidelity Figma prototype for a redesigned sleep-tracking app.

Outside of front-end work I'm comfortable with Python, SQL, REST APIs, and version control workflows, and I'm always picking up new tools. I approach every project with a product mindset: what does the user actually need, and what's the cleanest way to build it?

AI UX Critique Tool ↗

Upload any UI screenshot and receive an instant, structured heuristic evaluation powered by Claude. The tool analyses interfaces against Nielsen's 10 Usability Heuristics — flagging issues by severity (critical, major, minor) and surfacing actionable recommendations for each finding.

Built to bridge my UX research background with hands-on AI integration. Combines multimodal vision input, structured JSON outputs from the Claude API, and a clean React interface designed around the critique workflow itself.
